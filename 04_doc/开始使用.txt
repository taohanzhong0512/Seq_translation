【使用软件】
  1. 进入 "01_可执行文件" 文件夹
  2. 双击 "SEQ_to_PNG_Converter.exe"
  3. 选择 SEQ 文件，开始转换

【修改代码】
  1. 编辑 "02_源代码" 中的 Python 文件
  2. 测试: python seq_to_png_gui.py
  3. 满意后进行重新打包

【重新打包】
  1. 阅读 "README.md" 的打包部分
  2. 或查看 "04_文档说明/快速打包指南.md"
  3. 按照步骤复制文件到正确位置
  4. 运行 build_with_spec.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: 可执行文件能在其他电脑运行吗？
A: 能！复制 SEQ_to_PNG_Converter.exe 到任何 Windows
   64位电脑都可以直接运行，无需安装 Python。

Q: 如何修改界面或功能？
A: 编辑 02_源代码 中的文件，然后重新打包即可。
   详细步骤见 README.md 或快速打包指南。

Q: 打包需要什么？
A: 需要 Python 3.10+ 和 PyInstaller。
   运行 install_dependencies.bat 自动安装所有依赖。

Q: 图标如何修改？
A: 替换 05_资源文件/image 中的图标文件，
   然后重新打包即可。

Q: 软件太大了？
A: 91 MB 是正常的，包含完整的 Python 运行时和所有库。
   这样才能在没有 Python 的电脑上运行。
