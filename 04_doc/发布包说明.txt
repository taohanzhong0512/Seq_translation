Q: 可执行文件为什么这么大？
A: 包含完整的 Python 运行时和所有依赖库，
   这样才能在没有 Python 的电脑上运行。

Q: 可以只保留可执行文件吗？
A: 可以！如果只是使用，只需要
   SEQ_to_PNG_Converter.exe 一个文件。

Q: 如何修改程序？
A: 编辑 02_源代码 中的 Python 文件，
   然后参考文档重新打包。

Q: 需要安装什么软件吗？
A: 使用程序：不需要
   重新打包：需要 Python 3.10+

Q: 在其他电脑能运行吗？
A: 能！复制 exe 到任何 Windows 64位电脑
   都可以直接运行。
