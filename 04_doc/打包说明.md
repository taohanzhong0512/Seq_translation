## 📋 功能特性

### 文件选择
- 浏览并选择 SEQ 文件
- 自动生成输出目录名
- 自定义输出目录

### 参数配置
- 起始帧号（默认：0）
- 结束帧号（0 表示全部）
- 文件名前缀（默认：frame）

### 转换功能
- 支持 8/16/24 位深度
- 自动处理行填充字节
- 多线程转换
- 可随时停止

### 输出格式
- PNG 格式
- 文件名格式：`{prefix}_{frame_number:06d}.png`
- 例如：`frame_000000.png`, `frame_000001.png` ...

## 🐛 故障排除

### 如果程序无法启动

1. **检查系统兼容性**
   ```
   - Windows 7/8/10/11 (64位)
   ```

2. **尝试命令行运行**
   ```bash
   # 在 cmd 中运行，查看错误信息
   SEQ_to_PNG_Converter.exe
   ```

3. **检查杀毒软件**
   - 某些杀毒软件可能误报
   - 添加到白名单即可

### 如果转换失败

1. **检查 SEQ 文件**
   - 确保文件完整
   - 确保是 Norpix StreamPix 格式

2. **查看日志**
   - 界面底部有详细的错误日志
   - 根据错误信息调整参数
